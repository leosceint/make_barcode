# Делаем бинарный штрих-код

Состав:
- номер кадра 	(целое)
- дальность 	(целое)
- класс объекта	(целое)

**Штрих-код может быть произвольной длинны.** 
Задается длинна в конструкторе по сумме размерностей компонентов в него входящих

#### Пример штрих-кода на выходе (32 бит х 16 бит х 16 бит)

|         номер картинки         |     дальность    |     класс цели   |
|:------------------------------:|:----------------:|:----------------:|
|            5679                |        2345      |          4       |
|00000000000000000001011000101111| 0000100100101001 | 0000000000000100 |
